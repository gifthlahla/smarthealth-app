# SmartHealth App

> Mobile medical aid claims management system built with Flutter and Supabase.

SmartHealth allows users to create an account, submit medical claims with supporting documents (receipts, reports), and track claim statuses in real‑time.

---

## ✨ Features

- 🔐 Email/password authentication
- 📄 Submit claims for GP, Specialist, Medicine, Pathology, Hospital, Dental
- 📎 Upload documents to Supabase Storage
- 🔔 Real‑time claim status updates (Pending → Under Review → Approved → Rejected → Paid)
- 👤 User profile with membership number
- 🌙 Dark mode support
- 📱 Responsive design (iOS & Android)

---

## 🛠️ Tech Stack

- **Frontend**: Flutter (Riverpod, GoRouter)
- **Backend**: Supabase (PostgreSQL, Auth, Storage, Realtime)
- **State Management**: Riverpod (StateNotifier, StreamProvider)
- **File Picking**: image_picker, file_picker
